### [Design of Experiments course](https://github.com/archibald-carrion/Experiments-and-data-analysis)
> R, Rstudio

Repository containing code and data that I used in class of Design of Experiments (DoE), multiple dataset were used, goind from pokemon to network data. The goal of the experiments was to understand how to use DoE in different scenarios and how to analyze the data using different methods.

### [Pizzeria optimization](https://github.com/archibald-carrion/Pizzeria-optimization)
Repository containing an optimization project focused on maximizing the efficiency and profitability of a pizzeria. Using operations research techniques, mathematical modeling, and Python programming, it tackles real-world constraints like ingredient availability, production capacity, and business goals. Explore how optimization models can be applied to the food industry to drive smarter decisions and better outcomes.

### [Numerical Methods with Interactive Visualizations](https://github.com/archibald-carrion/Numerical-methods)
Repository containing a few example of interactive tools for learning numerical methods. It uses a variety of technologies, including Python, MATLAB, Julia, and more. Its a small collection of code I did in a numerical method course in college.

### [Clash of compilers](https://github.com/archibald-carrion/Clash-of-compilers)
Benchmarking C functions performance across GCC & Clang on Intel and AMD architectures using experimental design.


## 👾 AI & Machine Learning

### [Machine Learning course](https://github.com/archibald-carrion/Machine-learning-course)
My collection of machine learning and AI related projects, contains a wide variety of subject, like CNN clusters, neural networks, RAGs, etc.

### [Sauron decompiler](https://github.com/archibald-carrion/decompiler)
> machine learning, low level programming

Sauron Decompiler is a machine learning-powered project dedicated to the challenging task of decompiling binary executables back into human-readable C source code. Like the mythical phoenix rising from ashes, this project aims to reconstruct the original form and logic from its compiled representation.

### [The Restaurant - A Prolog/XPCE Project](https://github.com/archibald-carrion/Restaurant-food-composition-system)
Prolog-powered restaurant food composition system. It allows users to verify if a dish meets specific dietary needs by checking ingredient compatibility and restrictions. A simple, logic-based approach to personalized food recommendations.

### [Convolutional neural networks for sound detection](https://github.com/archibald-carrion/Convolutional-neural-networks-sound-classification)
Project using convolutional neural networks to detect sound of ambulance in audio.
